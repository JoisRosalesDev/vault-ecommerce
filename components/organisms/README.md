# Organisms (Organismos)

Los organismos son secciones y bloques funcionales complejos integrados por moléculas y/o átomos. Forman porciones distinguidas de la interfaz.

Ejemplos:
- Barra de navegación superior (`Navbar`)
- Cuadrícula de catálogo de productos (`ProductGrid`)
- Cajón del carrito de compras (`CartDrawer`)
- Formulario de edición/creación en el panel administrativo
